# 🌌 FocalCraft Launcher
> **「輕量、極速、美觀 —— 重新定義你的 Minecraft 遊戲管理體驗」**

FocalCraft Launcher 是一款基於 **Tauri v2 + Rust** 核心與 **React + TypeScript** 現代前端打造的新一代 Minecraft 啟動器。我們深知玩家在遊玩模組（Mod）與管理多個遊戲版本時的痛點，因此致力於提供一個**極致流暢、功能強大且視覺精美**的全新選擇。

---

## ✨ 核心特色亮點

### ⚡ 1. 極致輕量，將每一兆記憶體留給遊戲
* **Rust 核心驅動**：告別傳統啟動器動輒數百 MB 的記憶體佔用。FocalCraft 採用 Rust 作為高效能底層，結合 Tauri v2 渲染，軟體體積小、啟動在瞬息之間。
* **原生級流暢**：極低的系統資源消耗，即使在背景執行，也絕不影響您的遊戲幀率（FPS）。

### 🔌 2. 雙生態圈一鍵整合：Modrinth & CurseForge
* **內置一鍵搜尋與下載**：無需打開瀏覽器，在啟動器內即可直接瀏覽、搜尋並安裝 Modrinth 和 CurseForge 上的 Mod、光影包（Shaders）、資源包（Resource Packs）及世界存檔。
* **完美相容 `.mrpack`**：支援一鍵匯入 Modrinth 標準模組包（MRPack），自動解析並下載所有相依檔案。

### 🗃️ 3. 深度實例（Instance）沙盒化管理
* **完全獨立的環境**：每個遊戲實例（Instance）均為獨立沙盒，從 Mod、光影、存檔到伺服器列表（`servers.dat` 直接編輯器）皆互不干擾。
* **極客級設定**：支援為每個實例單獨配置 Java 路徑、記憶體分配、JVM 參數與自訂圖示。
* **資料夾即時監控**：內建資料夾監控功能，當您手動將檔案放入資料夾時，啟動器會即時更新狀態。

### 👕 4. 隨心所欲的「外觀衣櫃」（Wardrobe）
* **微軟 OAuth 安全登入**：採用微軟裝置碼登入安全協定，保障您的帳號安全，並支援離線模式。
* **本地外觀庫**：內建「外觀衣櫃」系統，允許您在本地收藏無限數量的皮膚（Classic / Slim），並支援一鍵同步上傳至 Mojang 官方伺服器與披風（Cape）快速切換。

### 🛠️ 5. 智慧 Java 環境託管
* **版本精準配對**：自動檢測當前 Minecraft 版本所需的 Java 版本。
* **一鍵自動下載**：如果系統缺少對應的 Java 運行環境，啟動器將會自動下載並配置，徹底擺脫「Java 版本錯誤」導致的無法啟動。

### 📊 6. 遊戲監控與即時主控台（Live Logs）
* **啟動狀態 HUD**：精心設計的啟動進度條與狀態顯示，讓您對遊戲加載進度瞭如指掌。
* **即時日誌系統**：內建 Live Log 主控台，即時輸出遊戲執行日誌，讓模組衝突或崩潰原因無所遁形。

## 🛠️ 技術優勢與架構

| 技術棧 | 扮演角色 | 優勢 |
| :--- | :--- | :--- |
| **Rust (Tauri v2)** | 後端核心 | 安全性高、執行效率極致、原生系統 API 呼叫、超小打包體積。 |
| **React + TypeScript** | 前端界面 | 模組化開發、嚴格的型別檢查，打造流動感、高反應率的 UI。 |
| **CSS Modules** | 視覺樣式 | 避免全域樣式污染，呈現細緻的現代毛玻璃（Glassmorphism）與微動畫效果。 |

## 💡 為什麼選擇 FocalCraft Launcher？

* **給 Vanilla 原版玩家**：乾淨俐落的介面，快速啟動，自動幫您下載合適的 Java。
* **給 Mod 狂熱玩家**：一鍵搜尋並下載 Modrinth/CurseForge 資源，自由開關、測試模組，實例之間井然有序。
* **給伺服器腐竹/社群**：自訂伺服器列表與特定模組包整合，讓隊友用最簡單的方式加入遊戲。
