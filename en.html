FEATURES

✨ Core Features

An experience tailored for players, free from bloated software and complex manual configuration.

⚡ Extremely Lightweight, Free Every Megabyte of RAM

Say goodbye to traditional launchers that consume hundreds of MBs of memory. Keep resources for what matters most: the game itself.

  • Rust Backend: Built on Tauri v2 rendering framework, extremely small footprint, launches in a flash.
  • Native Smoothness: Low background resource usage, ensuring maximum game FPS.

🔌 Dual-Ecosystem One-Click Integration

Integrates the two major platforms: Modrinth and CurseForge. No more opening browsers to download mods.

  • Built-in Search & Install: Directly search and install mods, shaders, resource packs, and worlds.
  • Full .mrpack Support: Supports one-click import of Modrinth modpacks with auto-resolving dependencies.

🗃️ Sandboxed Instance Management

Each game version and instance is completely isolated, avoiding version conflicts caused by mod clutter.

  • Independent Sandboxes: Mods, shaders, saves, and server lists (with built-in editor) never interfere.
  • Geek-level Configs: Customize Java paths, memory allocation, JVM arguments, and custom icons per instance.
  • Live Folder Monitoring: Launcher interface automatically detects and refreshes when files are dragged in.

👕 Flexible "Skin Wardrobe"

Perfect skin management module, providing local skin storage and one-click sync to Mojang.

  • Microsoft OAuth Login: Uses official secure device code flow, supports offline mode.
  • Local Wardrobe: Collect unlimited skins (Classic / Slim) and upload to Mojang servers in one click.
  • Cape Fast Switching: Easily manage and switch Mojang official capes.

🛠️ Smart Java Environment Hosting

Say goodbye to "Game crash on launch" due to mismatched Java versions.

  • Precise Version Matching: Automatically detects the best Java version required for the instance.
  • One-click Silent Download: Automatically downloads and configures Java if missing, no manual setup needed.

📊 Live Console & Crash Protection

Meticulously designed log and progress modules, making the game execution transparent.

  • Launch HUD: Beautiful progress bar and status HUD to show loading stages clearly.
  • Live Logs Console: Real-time game log streaming, making mod conflicts and crashes easy to diagnose.
