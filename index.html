FEATURES

✨ 核心特色亮點

為玩家量身打造的極致體驗，擺脫傳統啟動器的繁重與繁瑣配置

⚡ 1. 極致輕量，釋放每一兆記憶體

告別傳統啟動器動輒數百 MB 的記憶體佔用。將資源留給最重要的遊戲本體。

  • Rust 核心驅動：採用 Tauri v2 渲染架構，體積極小，瞬息啟動。
  • 原生級流暢：極低背景資源消耗，保證遊戲幀率（FPS）最大化。

🔌 2. 雙生態圈一鍵整合

整合 Modrinth 和 CurseForge 兩大主流生態。再也不需要手動開瀏覽器下載模組。

  • 內置一鍵搜尋下載：直接搜尋並安裝 Mod、光影、資源包及地圖。
  • 完整相容 .mrpack：支援一鍵匯入 Modrinth 標準包，自動下載所有相依。

🗃️ 3. 深度實例沙盒化管理

每一個遊戲版本與實例完全隔離。避免因為模組混亂導致的版本衝突。

  • 完全獨立沙盒：Mod、光影、存檔及伺服器列表（內建編輯器）互不干擾。
  • 極客設定檔：單獨配置 Java 路徑、記憶體大小、JVM 參數與自訂圖示。
  • 資料夾即時監控：手動拖入檔案時，啟動器界面將即時檢測並刷新。

👕 4. 隨心所欲的「外觀衣櫃」

完美的皮膚管理模組，提供本地皮膚存檔與一鍵官方同步。

  • 微軟 OAuth 登入：採用官方安全裝置碼協定，安全登入並支援離線模式。
  • 本地外觀衣櫃：收藏無限數量皮膚（Classic / Slim），一鍵上傳至官方。
  • 披風快速切換：快捷管理與更換 Mojang 官方披風。

🛠️ 5. 智慧 Java 環境託管

徹底告別因 Java 版本不符導致的「無法啟動遊戲」錯誤。

  • 精準版本配對：自動檢測當前 Minecraft 實例所需的最佳 Java 版本。
  • 一鍵靜默下載：如本機缺少對應 Java，軟體會自動下載並配置，無需手動干預。

📊 6. 即時主控台與崩潰防護

精心設計的日誌與進度模組，讓遊戲運行過程透明化。

  • 啟動進度 HUD：精美的 HUD 進度條與狀態顯示，清晰掌控載入環節。
  • Live Logs 主控台：即時串流遊戲輸出日誌，讓 Mod 衝突與崩潰原因無所遁形。
ARCHITECTURE

🛠️ 技術優勢與核心架構

我們放棄了傳統 Java 或 Electron 等繁重的開發技術，採用業界最尖端的語言與渲染架構，確保了極致的執行效能。

記憶體佔用對比 (MB) - 越小越好
FocalCraft (Tauri + Rust) ~45 MB
傳統啟動器 (Electron 核心) ~280 MB
官方啟動器 (Java Runtime) ~350 MB
Rust (Tauri v2)
後端核心：提供系統級記憶體安全保障、極致執行效率、原生系統 API 高速呼叫與極小化封裝體積（僅約 10MB）。
React + TS
前端界面：基於現代化 React 組件與 TypeScript 嚴格類型檢查，建構出反應極速、狀態流暢的高水準 UI/UX。
CSS Modules
視覺樣式：元件級樣式封裝，避免全域污染。渲染出細緻的現代毛玻璃（Glassmorphism）與滑順 of 微動畫交互。
